# 李睿軒的個人網站

這是我的個人網站專案，使用 Vue 3 + TypeScript + Tailwind CSS 開發。

## 專案特色

- 使用 Vue 3 Composition API 開發
- TypeScript 強型別支援
- Tailwind CSS 響應式設計
- 漸進式動畫效果
- 深色主題設計

## 頁面內容

- 首頁：個人簡介與技能展示
- 最新動態：分享近期活動與心得
- 常見問題：專案相關 FAQ
- 作品集：展示個人作品

## 本地開發

1. 安裝依賴

```sh
npm install
```

2. 啟動開發伺服器

```sh
npm run dev
```

## 部署

```sh
npm run build
```
